Module Hierarchy

  • xg: PyXG provides a python interface to Apple's Xgrid.

Class Hierarchy

  • xg.Connection: Track information needed to connect to an XGrid controller.
  • xg.GridManager: Manage the grids of a given Xgrid controller.
    • xg.Controller: A class for working with an Xgrid controller.
  • xg.InvalidIdentifierType: Invalid job or grid identifier type.
  • xg.Job: A class for working with an Xgrid job.
  • xg.JobManager: Manage a set of Xgrid jobs.
    • xg.Controller: A class for working with an Xgrid controller.
    • xg.Grid: A class for working with jobs on a specific Xgrid grid.
  • xg.JobSpecification: A class used for constructing multi-task batch jobs.
  • xg.XgridError: Xgrid exception class.
    • xg.InvalidAction: Invalide action exception.
    • xg.InvalidIdentifier: Xgrid exception for invalid job or grid identifiers.
      • xg.InvalidGridIdentifier: Invalide grid identifier exception.
      • xg.InvalidJobIdentifier: Invalide job identifier exception.
